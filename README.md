# CalendarAssistant
This is a project for Senior Project Summer of 2026. This is a group project made by Allison Arcand and Sonia Yahi

Front end and algorithm development and design has been handled by Allison Arcand 
We have set up the from end and machine learning 
For our high level architecture: 
The mobile app has been created on React Native
The Machine Learning has been set with python for predictions scoring and recommendations for future schedules 
For external APIs we have used Google Calendar API,and Microsoft Graph Calendar API

Back end and database development and OAuth 2.0 and design has been handled by Sonia Yahi 
For the backend setup and flow we are using Java, MongoDB and API gateway.

React Native with connect Java API with the google and microsoft aips and python machine learning. 

Structure:
App.tsx - where we will set up general navigation authorization and the maine theme from here you can open the authentication and login or the calendar itself. 
The login screen will hold the Google or Outlook connection buttons.

Front end setup:
Main app will be setup similarly to Google in having a homepage/dashboard to input new events, a new habit or the ability to edit old ones.
We will have a tab to show a full calendar this can be viewed by day week or month. 
Habits and recommendations page to show what your current habits are ( weekly phone calls or errands, daily skincare or gym sessions) and also will show you recommendations based on the past x amount of days (this will be changeable to account for students or other users that have changing schedules and habits with semesters and or kids schedules etc.) 
Settings and Profile tab. Here you can update the look of you calendar change your login or update other various settings.
