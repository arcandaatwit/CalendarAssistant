# CalendarAssistant
This is a project for Senior Project Summer of 2026. This is a group project made by Allison Arcand and Sonia Yahi
A full‑stack intelligent CalendarAssistant will be built with:
- React + Vite + FullCalendar (frontend)
- Node.js + Express + MongoDB + JWT (backend)
- Google Gemini AI (smart recommendations & insights)

This app helps users manage events, tasks, and routines while receiving AI‑powered scheduling suggestions based on productivity patterns.
